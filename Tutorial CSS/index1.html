Coisas que aprendi e achei interessantes:

  • CSS significa: Cascading Style Sheets (Folhas de Estilo Cascata);
  • É importante lembrar a estrutura CSS:
  • Os seletores simples podem ser de:
    1. Elementos;
    2. ID;
    3. Classes.
  • No seletor de classes, podemos configurar que apenas elementos específicas da classe sejam afetados. Exemplo: "p.red" (classe 'red' e elemento 'p');
  • Podemos utilizar o setor universal (*) quando quisermos afetar todos os elementos;
  • O css suporta 140 cores padrão;
