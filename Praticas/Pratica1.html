Lista de Exercícios 1

  1. Crie um arquivo HTML e CSS. Adicione um <div> e defina sua cor de fundo com background-color.
  2. No mesmo <div>, adicione uma imagem de fundo com background-image. Use uma imagem local ou uma URL.
  3. Modifique o estilo do <div> para usar background-repeat com os valores repeat, no-repeat, repeat-x, e repeat-y. Teste cada um.
  4. Aplique background-attachment: fixed; ao <div>. Adicione conteúdo para permitir a rolagem e veja a imagem fixada.
  5. Use background-position para posicionar a imagem de fundo. Experimente valores como top left, center, e bottom right.
  6. Combine todas as propriedades anteriores em uma única linha usando background no seu <div>.
  7. Crie um layout com <header>, <div> e <footer>. Aplique diferentes propriedades de fundo a cada um.

Lista de exercícios 2

  1. Crie uma página HTML com três caixas de texto (divs) empilhadas verticalmente. Use position: static para as caixas e observe o comportamento padrão do fluxo do documento.
  2. Crie uma caixa com position: relative e defina as propriedades top e left para 20px. Adicione um texto dentro da caixa e verifique como a posição do texto se ajusta em relação à posição original.
    3. Caixa
  3. Crie um contêiner com position: relative e dentro dele uma caixa com position: absolute. Defina a caixa absoluta para ter top: 10px e right: 10px. Observe como ela se posiciona em relação ao contêiner.
    4. Container1
    Caixa2
  4. Crie uma barra de navegação com position: fixed no topo da página. A barra deve permanecer visível ao rolar a página. Adicione alguns links fictícios.
    5. Barra de navegação
  5. Crie uma lista de itens com position: sticky e defina o valor top como 0. A lista deve permanecer no topo da tela ao rolar a página para baixo.
  6. Crie duas caixas com position: absolute. Uma deve ter um fundo vermelho e a outra azul, ambas com tamanhos iguais. Posicione a caixa vermelha em (20px, 20px) e a caixa azul em (30px, 30px) para demonstrar a sobreposição.
  7. Crie um menu dropdown com um botão. Use position: relative para o botão e position: absolute para o menu que deve aparecer abaixo do botão quando este for clicado. Ajuste a visibilidade do menu usando JavaScript ou CSS (hover).
