Lista de Exercícios 1

  1. Crie um arquivo HTML e CSS. Adicione um <div> e defina sua cor de fundo com background-color.
  2. No mesmo <div>, adicione uma imagem de fundo com background-image. Use uma imagem local ou uma URL.
  3. Modifique o estilo do <div> para usar background-repeat com os valores repeat, no-repeat, repeat-x, e repeat-y. Teste cada um.
  4. Aplique background-attachment: fixed; ao <div>. Adicione conteúdo para permitir a rolagem e veja a imagem fixada.
  5. Use background-position para posicionar a imagem de fundo. Experimente valores como top left, center, e bottom right.
  6. Combine todas as propriedades anteriores em uma única linha usando background no seu <div>.
  7. Crie um layout com <header>, <div> e <footer>. Aplique diferentes propriedades de fundo a cada um.
